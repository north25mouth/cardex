# Cardex テンプレート

## `cardex_ja.xlsx`

名刺管理用スプレッドシートのテンプレートです。

**このファイルを手動でセットアップする必要はありません。**  
Cardex にログインすると、このテンプレートと同じ構成のスプレッドシートが Google Drive に自動作成されます。

---

## シート構成

| シート名 | 内容 |
|----------|------|
| 名刺リスト | スキャンした名刺データが自動で追記されます |
| 使い方 | セットアップ手順と操作ガイド |

### 名刺リストの列

| 列 | 項目 |
|----|------|
| A | 登録日 |
| B | 氏名 |
| C | フリガナ |
| D | 会社名 |
| E | 役職 |
| F | メール |
| G | 電話番号 |
| H | 携帯番号 |
| I | 住所 |
| J | Webサイト |

---

## セットアップ手順

1. Cardex にアクセスし、Google アカウントでログイン
2. ログインと同時にスプレッドシートが自動作成される
3. アプリ内「設定」から差出人名・件名・本文テンプレートを設定して保存

以上で完了です。GAS（Google Apps Script）のデプロイは不要です。

---

## 旧バージョンについて

旧バージョン（GAS 連携）では `Code.gs` のデプロイが必要でしたが、現在のバージョンでは Google Sheets API / Gmail API を直接使用するため不要になりました。
